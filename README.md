# MailBox – Full Stack MERN Email App

MailBox is a full-stack email application built with the **MERN stack**. Users can securely send, receive, and manage emails. The app includes user authentication, inbox & sent folder management, composing emails, marking emails as read, and deleting emails.

---

## 🚀 Features

- **User Authentication:** Signup, Login, Logout using JWT.  
- **Compose Emails:** Send emails with subject, body, and recipient.  
- **Inbox & Sent Folders:** View received and sent emails in real-time.  
- **Mark as Read:** Easily mark emails as read/unread.  
- **Delete Emails:** Delete unwanted emails from inbox or sent folder.  
- **Responsive UI:** Clean and modern interface built with Tailwind CSS.  
- **State Management:** Redux Toolkit for user & mail state management.  
- **Backend APIs:** Express.js and MongoDB for persistent data storage.

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Redux Toolkit, Tailwind CSS, Axios, React Router  
- **Backend:** Node.js, Express.js, JWT Authentication  
- **Database:** MongoDB (via Mongoose)  
- **Other Tools:** dotenv, CORS
